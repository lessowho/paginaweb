¿Qué es un Protocolo Http?

Http es la abrieviatura de "Protocolo de transferencia de hipertexto". Permite realizar una petición de datos y recursos, como los documentos html. Es la base de cualquier intercambio de datos en la web.

¿Qué es un Query String?

Término que se utiliza para hacer referencia a una interacción con una base de datos y se encuentra presente en las url con un signo de interrogacion "?" para especificar su inicio.

¿Qué son los Verbos HTTP?

Son comandos de peticiones del programador, en un sistema REST son GET, POST, PUT, PATCH y DELETE.

¿Qué es un DNS?

ir El Sistema de nombres de dominio (DNS) es el directorio telefónico de Internet, sirve para que los navegadores puedan cargar los recursos de internet.

¿Qué es un CDN?

La Red de Distribución de Contenido (CDN) es un conjunto de servidores ubicados en distintos puntos alejados geograficamente, esto con la finalidad de mejorar la experiencia del usuario en la velocidad de acceso.
